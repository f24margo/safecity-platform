## [v1.0.0] - 2026-03-19

### ✨ Додано
- **Dashboard для менеджера безпеки:**
  - Головна сторінка з Security Score
  - KPI метрики (Score, Appeals, Monitoring)
  - Графіки динамики за тиждень
  - Переключення громад (Південне, Чорноморськ, Теплодар)
  - Багатомовність (UA/EN)

- **Сторінки Dashboard:**
  - Analytics — аналітика подій
  - Leaderboard — рейтинг громад
  - Moderation — модерація звернень
  - Settings — налаштування системи

- **Report Widget для жителів:**
  - 3-крокова форма звернення
  - Крок 1: Геолокація (GPS)
  - Крок 2: Завантаження фото
  - Крок 3: Опис події (макс 100 символів)

- **UI/UX:**
  - Темна тема (#0F172A / #1E293B)
  - Адаптивний дизайн
  - Gradient кнопки
  - Прогрес-бари

### 🐛 Виправлено
- TypeScript конфігурація
- React компоненти оптимізовані

### 📚 Документація
